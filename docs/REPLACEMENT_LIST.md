# 文件替换清单

## 可直接替换

- 用本项目根目录完整替换旧的 `PITETECH/` 目录。
- 不再需要旧的根目录重复 HTML 文件：`about.html`、`products.html`、`contact-us.html` 等。
- 不再需要 `assets/styles.css.backup`。

## 核心替换文件

- `assets/css/site.css`：替代 `assets/styles.css`
- `assets/js/site.js`：替代 `assets/main.js`
- `assets/images/`：替代 `assets/img/`
- `_redirects`：新增旧路由兼容
- `_headers`：替换原响应头配置
